==INSTALLATION==
The application needs to be installed before use the installation file is doinstall.php
In the Birkbeck instance run:
http://titan.dcs.bbk.ac.uk/~jbliss02/w1fma/doinstall.php
This re-creates the database so any existing information will be lost

===WEBSERVICE===

The web service for the application can be found at:
http://titan.dcs.bbk.ac.uk/~jbliss02/w1fma/webservice/imgdata.php';

if no url parameter is passed in then all images are returned
to get information about a specific file then append a variable path with the path of the image
e.g.
http://titan.dcs.bbk.ac.uk/~jbliss02/w1fma/webservice/imgdata.php?path=uploads/142860455360.jpg
